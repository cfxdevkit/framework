# @cfxdevkit/protocol

> Low-level protocol artifacts for tooling consumers: block explorers, indexers, MCP tools.
> Differs from `@cfxdevkit/contracts` by exposing non-curated, raw on-chain data without an opinionated TypeScript surface.

```bash
pnpm add @cfxdevkit/protocol
```

---

## What's included

- Event signatures and topic hashes for core Conflux contracts
- Function selectors for common protocol functions
- Raw ABI schemas for lower-level consumers
- Cross-space bridge event definitions
