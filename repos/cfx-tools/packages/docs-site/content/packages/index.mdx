## Dependency tree

```
@cfxdevkit/core                ← Tier 0 foundation
├── @cfxdevkit/protocol        ← cross-space bridge
├── @cfxdevkit/executor        ← automation primitives
├── @cfxdevkit/wallet          ← session keys (→ core, services)
├── @cfxdevkit/services        ← keystore, swap (→ core)
├── @cfxdevkit/compiler        ← Solidity tooling (standalone)
├── @cfxdevkit/contracts       ← ABIs/addresses (→ compiler)
├── @cfxdevkit/devnode         ← local node (standalone)
├── @cfxdevkit/react           ← React hooks (→ core)
├── @cfxdevkit/defi-react      ← DeFi hooks (→ core, services)
├── @cfxdevkit/wallet-connect  ← ConnectKit (→ wagmi)
└── @cfxdevkit/theme           ← design tokens (standalone)
```
