# CLI & Developer Tools

The `@cfxdevkit/cli` package provides a command-line interface (CLI) for interacting with Conflux developer tools. It supports commands for checking chain status, deriving wallet accounts, generating mnemonics, and extracting contract artifacts.

## Purpose

This module serves as the primary developer-facing CLI for the Conflux DevKit ecosystem. It offers a minimal, dependency-free parser for command-line arguments and exposes a set of developer utilities through a simple command structure.

## Key Components

### CLI Entry Point

The CLI is executed via `cfx` command, which parses arguments and routes to appropriate handlers.

```ts
// src/bin.ts
import { run } from './run.js';

const argv = process.argv.slice(2);
run(argv).then(
  (code) => {
    process.exit(code);
  },
  (err) => {
    process.stderr.write(`cfx: ${err instanceof Error ? err.message : String(err)}\n`);
    process.exit(1);
  },
);
```

A minimal argument parser handles flags, positional arguments, and command routing.


### Commands

#### `status`

Pings one or all known chains and reports their status.

```ts
// src/commands/status.ts
export async function runStatus(opts: RunStatusOptions): Promise<StatusReport[]>;
export async function statusFromFlags(flags: Record<string, string | boolean>, out: NodeJS.WritableStream): Promise<number>;
```

#### `derive`

Derives dual-space accounts (EVM and Core) from a mnemonic or generates a new one.

```ts
// src/commands/derive.ts
export function runDerive(opts: RunDeriveOptions): DeriveReport;
export function deriveFromFlags(flags: Record<string, string | boolean>, out: NodeJS.WritableStream): number;
```

#### `generate`

Generates a new BIP-39 mnemonic.

```ts
// src/commands/generate.ts
export function runGenerate(opts: RunGenerateOptions = {}): GenerateReport;
export function generateFromFlags(flags: Record<string, string | boolean>, out: NodeJS.WritableStream): number;
```

#### `contracts extract`

Extracts ABI and bytecode from Hardhat artifacts.

```ts
// src/commands/contracts.ts
export async function runContractsExtract(opts: RunContractsExtractOptions = {}): Promise<ContractsExtractReport>;
export async function contractsExtractFromFlags(flags: Record<string, string | boolean>, opts: RunOptions): Promise<number>;
```

## Usage Examples

```sh
# Check status of all chains
cfx status

# Check status of a specific chain
cfx status --chain core-testnet

# Generate a new mnemonic
cfx generate

# Derive accounts from a mnemonic
cfx derive --mnemonic "test test test test test test test test test test test junk"

# Derive 3 accounts starting at index 5
cfx derive --generate --count 3 --start 5

# Extract contract artifacts
cfx contracts extract --artifacts ./artifacts --out ./src/generated/contracts
```

## Scaffold CLI

The `@cfxdevkit/create` package provides project scaffolding capabilities.

### Templates

- `minimal-dapp`: Vite + React + @cfxdevkit/react frontend
- `wallet-probe`: Wallet detection and signing demo
- `project-example`: Full-stack project with frontend, backend, and contracts

### Usage

```sh
# Create a new project
npx @cfxdevkit/create my-project

# Use a specific template
npx @cfxdevkit/create my-project --template minimal-dapp

# Skip installation
npx @cfxdevkit/create my-project --skip-install
```
