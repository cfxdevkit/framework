# Test Coverage

Unit test coverage for packages in the `@cfxdevkit` monorepo.
Coverage is measured with [Vitest](https://vitest.dev/) and the **v8** provider.

Run coverage locally with:

```bash
# Single package (example)
pnpm --filter @cfxdevkit/cdk test:coverage

# All packages
pnpm exec moon run :test -- --coverage
```
